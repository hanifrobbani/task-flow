### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hanifrobbani/nextjs-boilerplate.git
   ```
2. Install Dependencies
   ```sh
   npm install
   bun install #recomended using bun
   ```
2. Start Local Development
   ```sh
   npm run dev
   bun run dev
   ```
